# Lista de Compatibilidade - R2NES V2

Este arquivo documenta o estado atual de compatibilidade das ROMs testadas no emulador.

### Legenda de Notas
- ⭐⭐⭐⭐⭐: Perfeito (Sem falhas conhecidas).
- ⭐⭐⭐⭐: Jogável (Pequenos glitches gráficos ou de áudio).
- ⭐⭐⭐: Médio (Bugs que afetam o gameplay, mas permitem jogar).
- ⭐⭐: Ruim (Muitos bugs, crasha ocasionalmente).
- ⭐: Não funciona (Não carrega ou crasha imediatamente).
